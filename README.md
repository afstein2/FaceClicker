# Face Clicker

## Description
Face Clicker is a simple incremental clicker game inspired by Cookie Clicker. The player clicks on a face to earn points, which can be spent in a shop to purchase upgrades that increase click power and speed.

As the player progresses, the face changes emotion and appearance based on the number of clicks earned.



## How to Play
Click on the face in the center of the screen to earn clicks. Use your clicks to buy upgrades in the store. Upgrades increase your click power and help you progress faster.

There are 5 progression levels. Each level changes the appearance and emotion of the face. Reach level 5 and unlock all upgrades to complete the game.



## Controls
- Left Mouse Click: Click the face to earn points
- Mouse: Navigate upgrade store UI


## Objectives
- Earn clicks by interacting with the face
- Purchase upgrades from the store
- Reach Level 5 by accumulating enough clicks and upgrades
- Fully evolve the face through all emotional states


## Features
- Click-based progression system
- Upgrade shop with multipliers
- Face emotion changes based on progress
- Particle effects on each click
- 5-level progression system
