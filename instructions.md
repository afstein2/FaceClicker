# Face Clicker

A cookie clicker like game


# Instructions
Create a simple game like cookie clicker, 
Use any assets online using an url for icons, fonts, game ui, etc
Do not download any images or files only use urls.
Create a UI for the store to buy differt power ups
Create the Clicker face, add particles each time you click (use the existing asset shapes)

## How to Play

The face is the cookie clicker its used to get the number of clicks. Use the clicks to buy power ups in order to get a faster click speed. and getting more clicks. Each amount of clicks the emotion and shapes of the face changes.
